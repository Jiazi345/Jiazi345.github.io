油脂笔显示与取色不一致

解决方案：
Render Properties ‣ Color Management set to Standard.


（可以用看到颜色不一致）
取色画上去总比色盘里的颜色偏淡。确认笔刷透明度设置正确后仍然没有改善
新建2D绘画项目   法线取色正常 查看官方文档后得知，2D项目做了设置：

  • World Properties ‣ Surface (Background) ‣ Color is set to white. 背景设为全白
  • Render Properties ‣ Color Management is set to Standard. 颜色模式设为stander
  • ..........
真正重要的是第二条颜色模式设置
